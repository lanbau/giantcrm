# giantcrm

## Installation Instructions
1. clone this repo
2. ```npm install --save```
3. ```bower install```
4. ```npm start```
5. visit http://localhost:5000 in your browser

## Technology Used
- Express
- Jade
- HTML
- CSS
- JavaScript
- Bootstrap

## APIs
- Google Contacts
- Google Calendar
- Gmail
- IBM Watson Personality Insights
